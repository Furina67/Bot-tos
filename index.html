Privacy Policy

Last updated: December 2025

1. Overview

This Privacy Policy explains how Furina collects, uses, and protects data when used in Discord servers. The bot operates only within servers where it is explicitly invited and does not function outside of Discord.

2. Data Collected

  • User, Server, and Channel IDs required for core functionality
  • Message content for moderation commands, AFK systems, counting channels, and deleted message sniping
  • Member data such as roles, nicknames, and join/leave state
  • Optional presence data for opt-in, status-based role features

The bot does not read or process direct messages.

3. How Data Is Used

  • Executing prefix commands and interactions
  • Moderation and server management
  • Utility tools, giveaways, and games
  • Optional AI-powered responses when explicitly requested by users

Data is not used for advertising, profiling, analytics, or training machine learning models.

4. Data Storage and Retention

  • Message content may be temporarily stored in a local JSON file only for moderation-related features such as deleted message sniping. Stored message content is limited in scope and automatically deleted after a maximum of 30 days.
  • AI command memory is stored only in short-term runtime memory to provide conversational context and is cleared automatically.
  • Configuration data (such as server settings) may be stored to support persistent functionality.

5. Data Sharing

The bot does not sell or share user data with third parties. AI-powered commands send user-provided input to a third-party AI service (Google Gemini) only when a user explicitly invokes an AI command.

6. User Control and Opt-Out

Server administrators can enable or disable features at any time. Users may request deletion of stored moderation-related data by contacting the bot developer.

7. Security

Reasonable technical and organizational measures are taken to protect processed data. The bot follows Discord’s Developer Terms of Service and API policies.

8. Changes to This Policy

This Privacy Policy may be updated as features evolve. Continued use of the bot after updates indicates acceptance of the revised policy.

9. Contact

For questions or concerns regarding this Privacy Policy, contact someone.1257 on Discord.
